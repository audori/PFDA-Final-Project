# Bendy Limbs Creation

## Repository
https://github.com/audori/PFDA-Final-Project.git

## Description
For my final project I will code a program that will create bendy limbs for a biped model. 
Whenever I rig models I usually take a lot of time figuring out bendy limbs so a program that automatically makes them for me would help me a lot.

## Features
- Creates the forearm joints
	- By selecting the elbow and wrist joint the code will create 4 joints in between the two
- Creates ribbon and CC (control curves)
	- Creates CCs on top of the joints created and ribbon
- Connect ribbon to CCs
	- Add influences to the ribbon to the CCs

## Challenges
- I will need to research how to code the joints to add more and effect their placement
- To add the joint influences to the ribbon you need to do that through skin weights and I'm not sure if you can code that
- I would be coding in MAYA so it might be a little different from VSCode

## Outcomes
Ideal Outcome:
- Creates a forearm bendy limb that is ready to paint the skin weights for

Minimal Viable Outcome:
- The ribbon, CCs, and joints are created.
