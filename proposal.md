# Stretcky IK Arm Creation

## Repository
https://github.com/audori/PFDA-Final-Project.git

## Description
For my final project I will code a program that will create squash and stretch limbs for a biped model. 
I'm going to be working on some rigs this summer so a tool to do this would make rigging the models way faster.

## Features
- Creates the forearm joints
	- By selecting the shoulder and wrist joint the code will create an Ik handle
- Creates locators and measures distance
	- Creates locators at selected joints and measures the distance between them to calculate how much the arm will squash and stretch
- Stretch and squash
	- the arm expands or shinks depending on how you move it

## Challenges
